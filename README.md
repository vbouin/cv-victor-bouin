# Victor Bouin — CV interactif

CV interactif (one-page, scrollytelling) — Design · Innovation · IA.
Identité Exoflow. 100 % statique, aucune dépendance build.

## Aperçu local
Ouvrir `index.html` dans un navigateur, ou :
```bash
python3 -m http.server 4680
```

## Publication GitHub Pages
Pousser sur une branche `main`, puis activer Pages (source : `main` / `/root`).
Le site sera servi à `https://<user>.github.io/<repo>/`.
